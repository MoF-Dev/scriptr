# Scriptr

Execute command-line scripts from a json file.

## Installation

```shell
npm -g install
```

## Usage

```shell
scriptr
```

## Flags
#### Add
```shell
scriptr -a
```
#### Delete
```shell
scriptr -d
```
### Edit
```shell
scriptr -e
```
### Help
```shell
scriptr -h
```
