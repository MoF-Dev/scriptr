# Scriptr

Execute command-line scripts from a json file.

## Installation

```shell
npm -g install
```

## Usage

```shell
node app.js
```

### Adding scripts

```shell
node app.js add --title="TITLEHERE" --script="SCRIPTCOMMANDS"
```
Separate commands with ;

### Deleting Scripts

```shell
node app.js delete
```

### Editing Scripts

```shell
node app.js edit
```
