# Scriptr

Execute command-line scripts from a json file.

## Installation

```shell
npm -g install
```

## Usage

```shell
scriptr
```

### Adding scripts

```shell
scriptr add --title="TITLEHERE" --script="SCRIPTCOMMANDS"
```
Separate commands with ;

### Deleting Scripts

```shell
scriptr delete
```

### Editing Scripts

```shell
scriptr edit
```
