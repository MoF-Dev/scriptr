# Scriptr

Execute command-line scripts from a json file.

## Installation

npm install

## Usage

```shell
node app.js
```

### Adding scripts

```shell
node app.js add --title="TITLEHERE" --script="SCRIPTCOMMANDS"
```

### Deleting Scripts

```shell
node app.js delete
```

### Editing Scripts

```shell
node app.js edit
```
